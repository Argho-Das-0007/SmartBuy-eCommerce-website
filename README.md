# SmartBuy - eCommerce Website (Frontend)

SmartBuy is a modern and responsive eCommerce web application interface designed to allow users to browse products, view product details, and manage shopping cart items through a user-friendly layout.
Note: This project currently includes frontend implementation only. Backend features such as authentication, database integration, and payment processing are planned for future development

## Features

- Product listing homepage
- Product browsing and detailed product view
- Shopping cart interface
- Responsive design for mobile, tablet, and desktop devices
- Clean and structured user interface
- Smooth navigation and interaction
- 
## Technologies Used

- HTML5  
- CSS3

## Getting Started

To run this project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/Argho-Das-0007/SmartBuy.git
   ```

2. Open the project folder

3. Open `index.html` in your browser
